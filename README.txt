SP70 Section 5 TRASHER Quiz PWA

Это не один HTML, а пакет PWA-приложения.

Как открыть на iPhone нормально:
1. Разместить папку sp70_pwa_app на HTTPS-хостинге: GitHub Pages / Netlify / Vercel / корпоративный сервер.
2. Открыть URL через Safari.
3. Нажать «Поделиться» → «Добавить на экран Домой».

Почему не работает при открытии файла:
iOS часто открывает HTML из Files/ChatGPT как Quick Look. В этом режиме JavaScript может быть заблокирован.

Логоблок TITAN-2 внутри приложения — стилизованная текстовая вставка, не официальный бренд-файл.
